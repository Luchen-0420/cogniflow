# 快速开始指南

## 3 步快速开始

### 步骤 1：加载插件

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的 **"开发者模式"**
4. 点击 **"加载已解压的扩展程序"**
5. 选择 `chrome-extension` 目录

### 步骤 2：登录账号

1. 点击浏览器工具栏中的**插件图标**
2. 在登录界面输入：
   - **用户名**：你的 Cogniflow 用户名
   - **密码**：你的密码
   - **API 地址**：默认 `http://localhost:3001/api`（可修改）
3. 点击 **"登录"** 按钮
4. 登录成功后会自动保存，无需再次登录

## 开始使用

### 方式一：快捷菜单栏（推荐）

1. 在任意网页中**选中文本**
2. 选中文本上方会**自动出现**快捷菜单栏
3. 点击菜单栏中的 **"笔记"** 选项即可保存

### 方式二：右键菜单

1. 在网页中**选中文本**
2. **右键点击**
3. 选择 **"保存选中内容到 Cogniflow"**

## 验证是否成功

保存成功后会看到：
- ✅ 右上角出现绿色提示："✓ 已保存到笔记"
- 快捷菜单栏自动消失

保存失败会看到：
- ❌ 右上角出现红色提示：错误信息

## 常见问题

**Q: 菜单栏不显示？**
- 确保已选中文本（不是空白）
- 菜单栏会显示在选中文本的上方，注意查看
- 刷新页面后重试

**Q: 保存失败？**
- 检查是否已登录（点击插件图标查看）
- 如果提示"未登录"，请先登录
- 检查 API 服务器是否运行（http://localhost:3001）
- 查看浏览器控制台的错误信息

**Q: Token 过期了？**
- 点击插件图标，退出登录后重新登录
- 或者直接重新登录，会自动更新 token

## 修改 API 地址

在登录界面可以修改 API 地址，修改后需要重新登录。
