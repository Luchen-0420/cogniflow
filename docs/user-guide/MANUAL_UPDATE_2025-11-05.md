#### 2.5 日程冲突检测

**冲突判定规则**：
```
日程A：14:00 - 16:00
日程B：15:00 - 17:00
→ 冲突！重叠时间：15:00 - 16:00
```

**视觉标识**：
- 🔴 红色边框
- ⚠️ 警告图标
- 💬 悬停提示

**解决方法**：
1. 调整时间
2. 取消日程
3. 标记例外

#### 2.6 重复性任务

**重复规则**：

- 📅 每月 - 可选日期或第N个星期X

**结束条件**：

- 📅 特定日期结束
- 🔢 重复N次结束

**管理操作**：

- 修改所有未来项
- 仅删除此项
- 删除整个系列
