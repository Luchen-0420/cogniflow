# 智能链接卡片优化 - 功能说明

## 📋 更新概述

为 CogniFlow 的链接管理功能进行了全面升级，新增 AI 智能梗概生成功能，并重新设计了链接卡片的视觉呈现。

## ✨ 核心功能

### 1. **AI 智能梗概生成**

#### 自动梗概
- 用户输入链接后，系统自动调用 AI 生成内容梗概
- 梗概长度：30-80 字，简洁有力
- 智能识别内容类型（GitHub、YouTube、博客文章等）

#### 手动生成
- 对于已保存的链接，可随时点击"生成梗概"按钮
- 支持基于用户备注生成更精准的梗概
- 生成失败时自动降级到基础描述

#### 智能分析
根据 URL 结构智能识别内容类型：
- **GitHub** → 识别为代码项目
- **YouTube** → 识别为视频内容
- **博客域名** → 识别为文章
- **文件扩展名** → 识别文档/图片类型

### 2. **全新链接卡片设计**

#### 视觉设计
```
┌─────────────────────────────────────────────────┐
│ 🌈 渐变装饰条                                    │
├─────────┬───────────────────────────────────────┤
│         │ 📌 链接标题                      [操作] │
│  [图标]  │ 🌐 域名                                │
│  缩略图   │ ✨ AI 生成的智能梗概                   │
│         │ 🔗 [访问链接] URL地址                  │
│         │ #标签1 #标签2                         │
│         │ 📅 创建时间 · 已抓取                   │
└─────────┴───────────────────────────────────────┘
```

#### 特色亮点

1. **渐变背景**
   - 青蓝色调渐变背景
   - 顶部彩色装饰条
   - 毛玻璃效果（backdrop-blur）

2. **大图标/缩略图**
   - 20x20 / 24x24 像素大尺寸显示
   - 圆角边框设计
   - 图片加载失败时优雅降级到默认图标

3. **清晰的信息层次**
   - 标题加粗，支持点击跳转
   - 域名小字显示，易于识别
   - 梗概突出显示，带 ✨ 图标标识

4. **交互优化**
   - 悬停放大效果（scale-102%）
   - 增强阴影
   - 按钮状态反馈

### 3. **技术实现**

#### 新增文件
- **`src/components/items/URLCard.tsx`** - 专门的 URL 链接卡片组件
- **`generateURLSummary()`** in `src/utils/urlProcessor.ts` - AI 梗概生成函数

#### 修改文件
- **`ItemCard.tsx`** - 检测 URL 类型时使用新的 URLCard 组件
- **`QuickInput.tsx`** - 添加自动生成梗概功能
- **`urlProcessor.ts`** - 新增 AI 梗概生成方法

#### 核心 API

```typescript
// AI 梗概生成
async function generateURLSummary(
  url: string,        // 链接地址
  urlTitle: string,   // 链接标题
  userContext?: string // 用户备注（可选）
): Promise<string>

// 示例
const summary = await generateURLSummary(
  'https://github.com/facebook/react',
  'React - A JavaScript library for building user interfaces',
  '学习前端框架'
);
// 返回：开源项目：构建用户界面的JavaScript库，由Facebook维护，拥有活跃的社区
```
