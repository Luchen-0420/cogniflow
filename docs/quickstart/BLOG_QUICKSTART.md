# 博客写作功能 - 快速开始

## 🚀 快速体验

### 第一步：打开编辑器
在输入框中输入 `/blog`，然后按回车或等待自动触发。

### 第二步：开始写作
编辑器会立即打开，你可以：
- 在左侧写作区输入 Markdown 文本
- 右侧实时预览效果
- 系统会在你停止输入 1 秒后自动保存草稿

### 第三步：保存文章
写完后，点击右上角的"保存并归类"按钮：
- AI 会自动分析文章内容
- 提取标题和标签
- 保存为笔记卡片
- 显示成功提示

## 📝 简单示例


**保存后**：
- 标题：`今天的收获`

## 💡 小贴士

1. **使用一级标题**：建议在文章开头使用 `# 标题`，这样 AI 可以直接识别
2. **草稿自动保存**：不用担心丢失内容，系统每秒自动保存
3. **字数统计**：顶部会显示实时字数，方便控制文章长度
4. **支持 Markdown**：所有标准 Markdown 语法都支持

## 🔧 快捷操作

在编辑器中：
- `Ctrl/Cmd + B`：加粗
- `Ctrl/Cmd + I`：斜体
- `Ctrl/Cmd + K`：插入链接
- `Shift + Enter`：输入框中换行（不提交）

## ❓ 常见问题

**Q: 草稿会保存多久？**
A: 草稿会一直保存在浏览器本地存储中，直到你清除浏览器数据或成功保存文章。

**Q: 可以同时写多篇文章吗？**
A: 目前只支持一个草稿，建议先保存当前文章再写下一篇。

**Q: AI 提取的标题不满意怎么办？**
A: 保存后可以在卡片详情中手动修改标题和标签。

**Q: 支持插入图片吗？**
A: 目前支持通过 Markdown 语法插入外部图片链接，未来版本会支持直接上传。
