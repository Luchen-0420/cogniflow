# 语音输入功能 - 快速开始

## 🎤 功能简介

语音输入功能让你可以通过说话的方式快速输入文字，无需打字，特别适合：
- 快速记录灵感
- 长篇文字输入
- 移动端使用
- 解放双手的场景

## 🚀 快速使用

### 1. 找到语音输入按钮

在主输入框左侧，你会看到一个**麦克风图标** 🎤

### 2. 开始语音输入

1. 点击麦克风按钮
2. 浏览器首次使用会询问麦克风权限，点击"允许"
3. 按钮变为红色并开始脉冲动画 = 正在录音
4. 对着麦克风说话
5. 说话的内容会实时显示在浮窗中
6. 再次点击按钮停止录音

### 3. 查看结果

- 识别的文字会自动出现在输入框中
- 可以继续编辑或直接发送

## 💡 使用技巧

### 提高识别准确率

1. **清晰说话**：语速适中，吐字清晰
2. **安静环境**：减少背景噪音
3. **靠近麦克风**：保持合适的距离
4. **使用标点**：说话时可以说"逗号"、"句号"等

### 最佳实践

```
✅ 好的使用方式：
"明天下午三点开会，讨论新产品发布方案"

❌ 避免的情况：
- 语速过快：明天下午三点开会讨论...（太快听不清）
- 环境嘈杂：背景有音乐或其他人说话
- 距离太远：离麦克风超过 50cm
```

## ⚠️ 常见问题

### Q1: 点击按钮没反应？
**A:** 检查以下几点：
- 浏览器是否支持（使用 Chrome/Edge）
- 是否允许了麦克风权限
- 麦克风硬件是否正常

### Q2: 识别不准确？
**A:** 尝试：
- 说话更清晰
- 减少背景噪音
- 使用更好的麦克风
- 检查网络连接

### Q3: 如何开启麦克风权限？
**A:** 
- **Chrome/Edge**：点击地址栏的锁图标 → 网站设置 → 麦克风 → 允许
- **Safari**：Safari 菜单 → 偏好设置 → 网站 → 麦克风 → 允许

### Q4: 可以识别英文吗？
**A:** 目前主要支持中文，英文和中英混合也能识别，但准确率可能稍低
